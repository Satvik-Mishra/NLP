# Sentiment and Keyword Analysis Tool

## Overview
This is a Streamlit-based tool for performing sentiment analysis and keyword analysis on text data. It allows users to either enter text directly or upload a document for analysis.

## Features
- **Sentiment Analysis:** Determines the sentiment (positive, negative, or neutral) of the input text.
- **Keyword Analysis:** Extracts common keywords from the text and displays their frequencies.

## Usage
1. **Installation:** Ensure you have Python installed. Clone this repository and install the required packages using the following command:
    ```
    pip install -r requirements.txt
    ```

2. **Running the App:** Execute the Streamlit app by running the following command in your terminal:
    ```
    streamlit run app.py
    ```

3. **Sentiment Analysis:**
    - Enter text directly into the provided text area.
    - Alternatively, upload a text document using the file uploader.
    - Click the "Show Document Text" button to display the content of the uploaded document.
    - The sentiment analysis results (sentiment, polarity, and subjectivity) will be displayed, along with a bar chart visualization.

4. **Keyword Analysis:**
    - Enter text directly into the provided text area.
    - Alternatively, upload a text document using the file uploader.
    - The common keywords and their frequencies will be displayed in a table.
    - Click the "Show Only Top 5" button to display only the top 5 keywords.
    - Additionally, a bar chart visualization of the top keywords will be shown.

## Implementation
1. **Integration:** Integrate the provided code into your Python project.

2. **Dependencies:** Ensure that you have the required dependencies installed (specified in `requirements.txt`).

3. **Run the App:** Execute the Streamlit app using the command mentioned above.

4. **Access the Tool:** Access the tool through the provided URL in your web browser.

5. **Input Data:** Enter text directly into the input fields or upload text documents for analysis.

6. **View Results:** Review the sentiment analysis results and keyword frequencies displayed by the tool.
